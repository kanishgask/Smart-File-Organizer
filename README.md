# Smart File Organizer

Smart File Organizer is a Java application that automatically organizes files into folders based on their type.

## Features

- Scans a folder
- Detects file type
- Moves files into categorized folders
- Automatically creates folders

## Tech Used

Java  
File Handling  
OOP Concepts

## Example

Before:

Downloads/
   image.jpg
   video.mp4
   notes.pdf

After:

Downloads/
   Images/image.jpg
   Videos/video.mp4
   Documents/notes.pdf

## How to Run

Compile:

javac *.java

Run:

java Main
